Pengertian Hosting

Hosting adalah layanan penyimpanan online untuk menempatkan file website agar dapat diakses melalui internet.

Contoh Hosting

  • Shared Hosting – hosting bersama, cocok untuk pemula.
  • VPS Hosting – server virtual dengan kontrol lebih besar.
  • Cloud Hosting – hosting memakai banyak server, lebih stabil.

Fungsi Hosting

  • Menyimpan file website (HTML, gambar, database).
  • Membuat website dapat diakses melalui internet.
  • Menjaga performa agar website tetap stabil.
